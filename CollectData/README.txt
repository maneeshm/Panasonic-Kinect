For Data Collection


Requirements
- Kinect camera
- GPS device & USB connector
- Headset & receiver
- Visual Studio
- Kinect SDK

Running

1.Double-click CollectData\CollectData\bin\Release\CollectData.exe
(Before running, make sure all devices are connected unless you have un-checked them in the dialog box)
2.Specify COM number of bluetooth GPS device
3.Hit Start & stop for length of data collection
4.Each time you want to collect, re-start from Step 1

Data collected can be found in separate folders in CollectData\CollectData\bin\Release
